Install the package from pip with:

```bash
pip install cesnet-datazoo
```

or for editable install with:

```bash
pip install -e git+https://github.com/CESNET/cesnet-datazoo
```

### Requirements
The `cesnet-datazoo` package requires Python >=3.10.
